### Working labs

If you want to follow my labs:

```bash
$ git clone https://github.com/William-Yeh/build-docker-with-ansible.git
$ cd build-docker-with-ansible
```

Case study subdirectories:

  - `lab-nginx` - Nginx

  - `lab-wrk` - Wrk


  - `lab-boom` - Boom

  - `lab-json-server` - JSON Server
